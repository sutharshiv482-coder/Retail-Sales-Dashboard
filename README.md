# Retail-Sales-Dashboard
Retail Sales Dashboard built using Excel, SQL, and Power BI. Sales data was analyzed to identify business insights and visualize key metrics such as sales, profit, profit margin, product performance, and regional trends through an interactive dashboard.

# Retail Sales Dashboard

## Project Overview
This Power BI dashboard provides insights into retail sales performance, profit, customer behavior, and product trends.

## Objectives
- Track sales performance
- Monitor profit margin
- Analyze top-selling products
- Identify loss-making products
- Compare regional performance

## Tools Used
- Power BI
- SQL
- Excel

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin %
- Total Orders
- Average Sales per Order

## Dashboard Features
- Interactive filters and slicers
- Sales trend analysis
- Category-wise performance
- Regional analysis
- Top and bottom products

## Insights
- Technology category generated the highest profit.
- Some products generated sales but resulted in losses.
- West region achieved the highest sales.
